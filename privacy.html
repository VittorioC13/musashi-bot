Privacy Policy

Last Updated: February 25, 2026

TL;DR - The Important Stuff

  • We don't collect or store your personal data
  • No user accounts, no tracking, no analytics
  • All processing happens locally in your browser
  • We only send data to DeepSeek AI when you explicitly request sentiment analysis
  • 100% free, no ads, no data monetization

1. Introduction

Musashi ("we", "our", or "us") is a Chrome browser extension that provides AI-powered sentiment analysis and prediction market signals on Twitter/X. This Privacy Policy explains how we handle data when you use our extension.

2. Information We Collect

2.1 Data We DO Collect

Data Type Purpose Storage Location
Prediction market data (titles, odds, URLs) Match markets to tweets and display current odds Local browser storage only
Tweet content (when you request analysis) Perform AI sentiment analysis Temporarily sent to DeepSeek API, not stored
User preferences (e.g., theme settings) Remember your extension settings Local browser storage only

2.2 Data We DO NOT Collect

  • Your Twitter/X username or account information
  • Your browsing history
  • Your personal information (name, email, address, etc.)
  • Your IP address or location
  • Analytics or tracking data
  • Cookies or fingerprinting data

3. How We Use Data

3.1 Tweet Content Processing

When you browse Twitter/X, Musashi:

  1. Reads tweet text to automatically detect mentions of prediction markets
  2. Displays matching markets from Polymarket and Kalshi directly on tweets
  3. Does NOT send tweet content anywhere unless you explicitly click to analyze sentiment

3.2 AI Sentiment Analysis (Optional)

When you explicitly request sentiment analysis by clicking the "Analyze" button:

  1. The tweet or selected text is sent to DeepSeek API for AI analysis
  2. DeepSeek returns sentiment analysis (Bullish/Bearish/Neutral) with confidence scores
  3. Results are displayed to you and cached locally for performance
  4. We do NOT store your analysis requests on our servers (we don't have servers!)

Important: DeepSeek AI Processing

When you request sentiment analysis, tweet content is sent to DeepSeek's API. DeepSeek's privacy policy applies to this data processing. DeepSeek may use this data to improve their AI models. You can review DeepSeek's privacy policy at: https://www.deepseek.com/privacy

3.3 Local Data Storage

Musashi stores the following data locally in your browser:

  • Market cache: Current prediction market odds (updated every 30-60 seconds)
  • Analysis cache: Recent sentiment analysis results (to avoid repeated API calls)
  • User preferences: Your extension settings

This data:

  • Never leaves your device
  • Is NOT transmitted to our servers (we don't have servers!)
  • Can be cleared by uninstalling the extension or clearing browser data

4. Third-Party Services

4.1 DeepSeek AI

Purpose: Sentiment analysis of tweets and news

Data shared: Tweet text or selected text (only when you request analysis)

Privacy policy: https://www.deepseek.com/privacy

4.2 Polymarket API

Purpose: Fetch prediction market data (titles, odds, URLs)

Data shared: None - we only fetch public market data

Privacy policy: https://polymarket.com/privacy

4.3 Kalshi API

Purpose: Fetch regulated prediction market data

Data shared: None - we only fetch public market data

Privacy policy: https://kalshi.com/privacy

5. Data Sharing

We do NOT sell, rent, or share your data with third parties for marketing purposes.

The only data sharing that occurs:

  • Tweet content sent to DeepSeek API only when you explicitly request analysis
  • Public API requests to Polymarket and Kalshi to fetch market data (no personal data involved)

6. AI Agent API Access

Musashi provides an API for AI trading agents to access sentiment analysis. When your AI agent uses Musashi:

  • The AI agent runs on YOUR machine and queries YOUR installed Musashi extension
  • No data is transmitted to us (we don't have servers!)
  • The same privacy rules apply - data is only sent to DeepSeek when analysis is requested
  • You control what your AI agent does with Musashi's data

7. Data Retention

  • Market data cache: Updated every 30-60 seconds, older data automatically deleted
  • Sentiment analysis cache: Kept for 24 hours, then automatically deleted
  • User preferences: Kept until you uninstall the extension or clear browser data

8. Your Rights

You have the right to:

  • Access your data: All data is stored locally in your browser - you can inspect it using Chrome DevTools
  • Delete your data: Uninstall the extension or clear browser storage data
  • Opt-out of AI analysis: Simply don't click the "Analyze" button - sentiment analysis is entirely optional
  • Control data sharing: You decide when to request sentiment analysis (which sends data to DeepSeek)

9. Children's Privacy

Musashi is not intended for users under the age of 13. We do not knowingly collect data from children under 13. If you are a parent or guardian and believe your child has used Musashi, please contact us.

10. Security

We take security seriously:

  • All API requests use HTTPS encryption
  • No server-side storage (can't be hacked if we don't store it!)
  • All data processing happens locally in your browser
  • Extension code is open for security audits

11. Changes to This Policy

We may update this Privacy Policy from time to time. When we do:

  • The "Last Updated" date at the top will change
  • We'll notify users via the extension (if changes are significant)
  • Continued use of Musashi after changes means you accept the new policy

12. International Users

Musashi can be used globally. However:

  • DeepSeek AI may process data in various countries
  • No personal data is collected or stored by Musashi itself
  • GDPR compliance: We don't collect personal data, so GDPR data subject rights don't apply
  • CCPA compliance: We don't sell personal information

13. Contact Us

14. Legal Compliance

GDPR Compliance (European Users)

Under GDPR, we are a "data processor" only when you request sentiment analysis (which sends data to DeepSeek, the "data controller"). Since we don't collect or store personal data ourselves, most GDPR obligations don't apply to Musashi directly.

CCPA Compliance (California Users)

We do NOT sell your personal information. We do NOT share personal information for cross-context behavioral advertising.

Chrome Web Store Requirements

This privacy policy complies with Chrome Web Store's Program Policies and User Data FAQ.
