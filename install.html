Troubleshooting

Extension Not Working?

  • Make sure you extracted the ZIP file (don't load the .zip directly)
  • Select the folder containing manifest.json
  • Refresh Twitter/X after installation
  • Check Chrome console for errors (F12)

No Markets Showing?

  • Wait 10-15 seconds after loading the extension
  • Go to chrome://extensions ‚Üí Musashi ‚Üí "service worker"
  • Look for "[Musashi SW] Stored X markets" message
  • Refresh Twitter/X page
